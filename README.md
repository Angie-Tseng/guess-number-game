# 猜數字遊戲

這是一個具有UI的猜數字遊戲，使用Python進行開發。

### 初始階段

玩家可自行設定數字的範圍，按下開始後，系統會從設定的範圍中抽出隨機一個數字給玩家猜，預設範圍為1~100。

### 猜數字階段
玩家可以從當前的範圍中隨意猜一個數字，若沒猜中，則系統會以所猜的數字作為新的上界或下界，重複進行直到猜對為止。


### 舉例
- 初始範圍1~100，答案為28
- 玩家猜50，範圍縮至 1 ~ **50**
- 玩家猜25，範圍縮至**25** ~ 50
- 玩家猜37，範圍縮至25 ~ **37**
- 玩家猜31，範圍縮至25 ~ **31**
- 玩家猜28，遊戲結束
